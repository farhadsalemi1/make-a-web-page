Lesson 1: The Basics of the Web and HTML

The basics

In this lesson we learned the Basics of HTML programming. HTML stands for Hyper Text Markup Language. We learned about The Web, HTML, URL. HTTP, and Web applications.

HTML Vs HTTP

HTML is the Document and is the plain text, and HTTP is the protocol.

Tags

We learned Tags and Elements. We also learned about Inline and block concept. Some of the tags we have learned are: b, em, a, img, br, p, div.
Tags can be void which means there wouldn’t be any closing tag fro them. Tags can have attributes.

Global Attributes

Some of the attributes are global which means they can be uses in any tags.

Coding Applications

We also learned about Scratchpad, Codepen and Github applications.

Lesson 2: Creating a Structured Document with HTML

Tree Structure

In this lesson we learned about the tree structure of a HTML page, which means we have tags and we have texts and tags inside those tags.

DOM: Document Object Model, the tree-like structure of a page.

HTML Vs CSS

HTML and CSS are both Languages. HTML controls the structure of a web page. CSS controls the style of a page (how it looks).

Boxes, boxes everywhere.

In HTML every area of the page we are making, belongs to a box. so the whole page can be divided into multiple boxes. We define these boxes with the “div" tag. We use class=attribute to give the boxes different styles.
